C Source Events

Source events are delivered to the application's source event callback lbm_src_cb_proc, which is set in the lbm_src_create() call. The event type is passed as the "event" parameter.

Here are the different event types (click each one for a full description). Note the list appears twice; first sorted by the event number, then sorted by event name.
