Detailed Description

Structure that holds source wakeup event data.

This structure is delivered to the lbm_context_src_cb_proc or lbm_src_cb_proc callbacks in their "ed" parameters when the "event" parameter set to LBM_SRC_EVENT_WAKEUP.

See lbm_src_event_wakeup_t_stct for field definitions.
